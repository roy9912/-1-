    조건부서식

    -제목 필드 설정 X
    -무조건 처음 열의 값이 비교 대상으로 포함되어야 한다!!
    -f4로 부분 절대화
    -모르는 함수가 있으면 한번 적어보고 범위 설정 들어가자
    -잘 나오는 함수정리
    Left, right, mid 사용 시 “” 또는 value 또는 *1 사용
    Rank 사용 시 상위는 rank(X,X,0), 하위는 rank(X,X,1)
    Countif 사용 시 ex) 김/이씨 성을 가진 인원수가 20 이상인 행 countif(X,”>=20”)=2
    시트보호 / 페이지설정
    -시트보호 = 한 시트 내 수정 X
    -통합문서보호 = 시트 삽입, 삭제, 이름 바꾸기 X
